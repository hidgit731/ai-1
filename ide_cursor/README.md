# Анкета профориентации

Минимальное приложение: простой HTTP-сервер на Python без фреймворков и статический фронтенд на vanilla JS. Запуск через Docker работает.

## Запуск через Docker Compose

```bash
docker-compose up --build
```

Откройте в браузере `http://localhost:8000/`.

## Запуск локально без Docker

Требуется Python 3.11+.

```bash
python app.py
```

## Что внутри

- `app.py` — минимальный HTTP-сервер, отдаёт `index.html`, API `GET /questions`, `POST /answers` (ответы хранятся в памяти процесса).
- `index.html` — фронтенд: загружает вопросы, генерирует форму, отправляет ответы и показывает alert «Спасибо!».
- `docker-compose.yml`, `Dockerfile` — для локального развёртывания. Экспонируется порт `8000`.
