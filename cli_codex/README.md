# Анкета профориентации

Простое монолитное приложение для заполнения анкеты на тему профориентации. Backend реализован на Python на базе стандартной библиотеки `http.server`, фронтенд на чистом HTML/CSS/JS. Все данные хранятся в памяти процесса.

## Локальный запуск

1. Установите Docker и Docker Compose.
2. Соберите и запустите проект:
   ```bash
   docker compose up --build
   ```
3. Откройте браузер по адресу [http://localhost:8000](http://localhost:8000) и заполните анкету.

Docker Compose поднимает один контейнер, в котором работают и API, и статические ресурсы.
